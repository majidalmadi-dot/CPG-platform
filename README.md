# KSUMC National CPG Authority Platform

**المرجع الوطني للأدلة الإكلينيكية**

AI-powered Clinical Practice Guideline development platform for King Saud University Medical City, functioning as a national guideline authority for Saudi Arabia.

## Features

- **5-Tier Governance Model** — NSC → TOC → GDG → ERT → External Review
- **10-Stage GRADE Workflow** — Topic selection through publication
- **5 Integrated AI Skills** — Evidence-to-Guideline, SR/MA, CEA, Health Policy Scanner, PHPSA
- **Live Evidence Search** — PubMed, Cochrane, CrossRef API integration
- **Role-Based Access** — 10 authority levels with Supabase authentication
- **GRADE Engine** — Automated certainty calculation with AGREE II quality gates

## Frameworks

GRADE · NICE · GIN-McMaster · AGREE II · ADAPTE

## Quick Start

1. Clone this repository
2. Open `index.html` in a browser — the platform works immediately
3. For backend: see `DEPLOY.md` for Supabase setup instructions
